# GitHub 偵察機 🔍

一個用 Vue 3 做的網頁應用，用來搜尋 GitHub 倉庫。

## 功能

- 🔍 搜尋 GitHub 專案
- 📋 顯示專案名稱、描述、星數
- 👤 顯示專案擁有者的大頭照
- 🌙 深色主題介面
- ⚠️ 查無此人提示

## 技術

- Vue 3
- Vite
- GitHub REST API

## 啟動方式

```bash
npm install
npm run dev
```

## 📚 Vue 重要指令的生活比喻

### 🔗 v-model = 「雙向繩子」

想像一下...
你有一個**遙控器**（輸入框）和一個**電視螢幕**（變數盒子）。

- 遙控器按「+」 → 電視頻道跳到 10
- 電視遙控器按「-」 → 頻道跳到 5

**v-model** 就是那條連接遙控器和電視的**隱形繩子**：

```html
<input v-model="searchName" />
```

> 你在輸入框打 `vue` → `searchName` 盒子立刻變成 `vue`
> 如果 `searchName` 盒子變成 `react` → 輸入框也跟著變成 `react`

---

### 🔌 v-if = 「電源開關」

想像一下...
你家有一個**電燈**（畫面上的文字或元素）。

- 開關打開（true）→ 燈亮 → 看得見
- 開關關掉（false）→ 燈滅 → 看不見

**v-if** 就是那個**電源開關**：

```html
<p v-if="loading">打電話中...</p>
```

> 當 `loading` 是 `true`（正在打電話）→ 顯示「打電話中...」
> 當 `loading` 是 `false`（電話打完）→ 這行字消失

---

### 📝 對比一下

| 指令 | 比喻 | 做什麼 |
|------|------|--------|
| `v-model` | 雙向繩子 | 把 A 和 B 連在一起，兩邊都會跟著變 |
| `v-if` | 電源開關 | 控制「顯示」或「隱藏」 |
